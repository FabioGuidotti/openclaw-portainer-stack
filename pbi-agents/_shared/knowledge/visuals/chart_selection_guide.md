# Chart Selection Guide

## Choose by Data Question

| Question | Best Chart | Avoid |
|----------|-----------|-------|
| How much? (comparison) | Bar/Column | Pie (>5 items) |
| What's the trend? | Line | Bar |
| What's the proportion? | Stacked bar, Treemap | Pie (>5 items) |
| What's the distribution? | Histogram, Box plot | Bar |
| What's the relationship? | Scatter | Line |
| What's the composition? | Stacked area, Waterfall | Pie |
| What's the performance? | KPI card, Gauge | Table |
| Where? (geography) | Map | Bar |

## Power BI Visual Recommendations

| Visual | Max Items | Best For |
|--------|-----------|----------|
| Card | 1 value | KPIs, totals |
| KPI | 1 value + trend | Target tracking |
| Bar Chart | 5-15 items | Comparison |
| Line Chart | Time series | Trends |
| Table | Any rows | Detail data |
| Matrix | Hierarchies | Drill-down |
| Treemap | 10-20 items | Part-to-whole |
| Scatter | 50+ points | Correlation |
| Map | Geographic | Location data |

## Rules

1. Maximum 7-8 visuals per report page
2. Most important KPIs at top-left (reading order)
3. Related visuals should be spatially grouped
4. Use consistent color coding across pages
