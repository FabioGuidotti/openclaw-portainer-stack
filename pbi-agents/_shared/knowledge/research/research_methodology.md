# Research Methodology

## Source Priority

1. **Microsoft Official Docs** (learn.microsoft.com) — highest authority
2. **SQLBI** (sqlbi.com) — DAX and modeling best practices
3. **Power BI Blog** (powerbi.microsoft.com/blog) — feature announcements
4. **Community Forums** — real-world patterns (verify before adopting)
5. **YouTube/Blogs** — useful but always cross-reference

## Research Output Format

```markdown
## Research: <Topic>

### Summary
<2-3 sentence summary>

### Key Findings
1. <finding with source>
2. <finding with source>

### Sources
- [Title](URL) — accessed YYYY-MM-DD
- [Title](URL) — accessed YYYY-MM-DD

### Applicability
<How this applies to the current project>

### Confidence
<High|Medium|Low> — based on source quality and recency

### Recommendations
1. <actionable recommendation>
```

## Rules

- Always include access date for URLs
- Distinguish between facts and opinions
- Flag version-dependent information (e.g., "requires PBI Desktop March 2024+")
- Note when information may be outdated
